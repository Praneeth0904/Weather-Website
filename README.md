# 🌦️ Weather.io – Weather Web Application

A responsive weather application that provides real-time weather data and forecasts based on user location.
Built to deliver accurate information with a clean and user-friendly interface.
Designed to work seamlessly across all devices.

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript (Vanilla)

---

## ✨ Features

* Real-time weather updates based on user location
* 7-day and hourly weather forecast
* Displays temperature, humidity, and precipitation data
* Responsive and user-friendly interface
* Location detection using geocoding

---

## 🔗 APIs Used

* OpenWeatherMap API (weather data)
* Geocoding API (location-based data)

## ⚙️ How It Works

1. Detects user location using geocoding
2. Fetches weather data from external APIs
3. Parses and displays data in a clean UI

---

## 🧠 Challenges Faced

* Integrating multiple APIs smoothly
* Parsing and formatting complex weather data
* Handling API errors and invalid responses

---

## 🐛 Common Errors Handled

* Invalid API key (401 error)
* Unauthorized access (403 error)
* Null element reference issues in JavaScript
* Incorrect location detection
